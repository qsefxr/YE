# YE | The Visionary

## Обзор проекта
Данный проект представляет собой интерактивный одностраничный веб-сайт (landing page), посвящённый творческому пути и дискографии Канье Уэста (Kanye West). Сайт создан с упором на современный, премиальный дизайн, глубокое погружение пользователя и сложный сторителлинг, используя продвинутые анимации и эффекты при прокрутке.

## Ключевые особенности
- **Интерактивность и анимации:** Интенсивное использование библиотеки GSAP (ScrollTrigger, ScrollToPlugin) для создания плавных анимаций при скролле.
- **Горизонтальный скролл:** Секция "The Bear Trilogy" реализована в виде горизонтальной прокрутки, которая активируется при вертикальном скролле пользователя.
- **Кастомный курсор:** Интерактивный кастомный курсор с плавным следованием (follower) и специальными состояниями при наведении на кликабельные элементы.
- **Визуальные эффекты:**
  - Глитч-эффекты для текста.
  - Анимированный Mesh-градиент на фоне, меняющий цвета в зависимости от текущей секции.
  - Эффект матового стекла (Glassmorphism) для карточек и элементов интерфейса.
  - 3D-трансформации (perspective) и эффекты параллакса для изображений.
- **Генератор цитат:** Интерактивный блок в конце страницы, позволяющий по клику менять случайные цитаты.
- **Адаптивность:** Полная поддержка мобильных устройств с адаптивным меню и изменением логики скролла (переход от горизонтального скролла к вертикальному на мобильных устройствах).

## Технологический стек
- **HTML5:** Структура и семантика.
- **Tailwind CSS:** Использован через CDN для быстрой стилизации, задания типографики и цветовой палитры.
- **Vanilla CSS:** Пользовательские анимации (keyframes), стилизация скроллбара, кастомного курсора, глитч-эффектов и сетчатых фонов.
- **Vanilla JavaScript:** Логика работы страницы.
- **GSAP (GreenSock Animation Platform):** Используется для всех сложных анимаций на странице.

## Оптимизация
Сайт был дополнительно оптимизирован для улучшения производительности, особенно на ПК:
- **Оптимизация курсора:** Анимация кастомного курсора была переведена с использования свойств `top`/`left` на использование `x`/`y` (CSS Transforms). Это позволяет избежать постоянных перерисовок (reflow/repaint) макета при движении мыши и задействует аппаратное ускорение (GPU), значительно снижая нагрузку на процессор и убирая лаги.
- **Чистый код:** Из кода (HTML, CSS, JS) были удалены все комментарии для уменьшения размера файлов.

## Установка и запуск
1. Склонируйте или скачайте репозиторий.
2. Откройте файл `index.html` в любом современном браузере. Никакой дополнительной сборки или установки зависимостей не требуется (Tailwind и GSAP загружаются через CDN).
